# 🚀 AI-Powered Resume Screening & Career Recommendation System

## 📌 Overview
This project is a web-based AI-powered Resume Screening and Career Recommendation System developed using Python and Flask.

It uses Natural Language Processing (NLP) techniques such as TF-IDF and Cosine Similarity to:

- Calculate resume-job match percentage
- Suggest best-fit career roles
- Provide intelligent career recommendations

---

## 🧠 Technologies Used

- Python
- Flask
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity
- PyPDF2
- Bootstrap 5
- HTML & CSS

---

## 🎯 Features

- Upload Resume (PDF)
- Paste Job Description
- AI-based Match Score Calculation
- Multi-Domain Career Role Recommendation
- Animated UI with Loading Effects
- Professional Web Interface
